DreamCSV (CSV importer/exporter)

Purpose

This script was made to help DreamHost users import and export data from the dreamhost API into csv files, which they can modify offline and then re-import.

Oh, and the DreamHost API contest

Why?

To be totally honest, I hate CSV, and I hate using spreadsheets. But, From time to time, it is the easiest way to communicate with people. There are many advantages to using CSV, as many of you know, you can load CSV files as tables into Excel. Do you already have lists of email addresses in spreadsheets? Business contacts? DNS records? Is someone in your organization manually entering this data into dreamhost?

In many cases, it may be easier to to all these updates in Excel, and sync the updates with DreamHost

How it works

The CSV importer is a "generic" importer, meaning that you can use it with any of the dreamhost API features. There are 2 main modes of operation:
  • send (imports a CSV file into dreamhost, can delete stuff too!)
    • e.g. ./dreamcsv --send data.csv --cmd dns-add_record
    • e.g. ./dreamcsv --send data.csv --cmd dns-remove_record
    • e.g. ./dreamcsv.pl --send list1.csv --cmd announcement_list-add_subscriber --listname list1 --domain joshlange.net
  • recieve (exports data from dreamhost into a CSV)
    • e.g. ./dreamcsv --receive data.csv --cmd dns-list_records
Forgot the name of the command you want to use? Yeah, they are a bit tedius
to remember.....
But, thats not a problem! just omit --cmd, and you will be reminded what you have access to.
  • ./dreamcsv.pl --send list1.csv
DreamCSV doesn't check to make sure a record doesn't exist before it tries
to write it. But, this should be ok, because the dreamhost API commands that
I've tried fail gracefully. What do I mean? well....Try adding a dns record that
exists. No harm done.

Just make sure you don't delete anything important ;-)

Supported Platforms

Any platform that supports perl. This includes Windows, Linux, and MacOS.

Getting Started

The easiest way to get started with DreamCSV is to download it, and run it on the command line.
  • Install perl from your operating system's package manager, if you don't already have perl
  • IMPORTANT: install Text::CSV_XS (NOT Text::CSV)
    • Available in Windows' Active Perl package manager
    • Available from most linux distro's package managers
    • Available from cpan.org, in the unlikely case you have to install it manually
  • If you are using windows, install ActivePerl for Windows
  • If you get a "bad interperter" error, put the correct path to perl on line 1 of the script

Oh, and if you are really doing subscription list management, I created a few sample batch files for you.

Once you input the proper values, and put them on a windows desktop, you can send/receive
email lists from the dreamhost servers by double clicking!
  • Great....? For the office administrator, who's in charge of keeping your organization's mailing lists up to date!

Setup

  1. Install perl
    • You probably already have it.
    • On Windows, install ActivePerl
  2. Generate an API key
    • Make sure that everything you want to use is checked
    • Uncheck everything else
  3. Edit configuration (so you don't need to give cli arguements, optional)
    • In a text editor, put your api key, username, and into the script where indicated.
  4. Test out the script
    • Open a terminal, and run the script: ./dreamcsv.pl --receive test.csv --cmd api-list_accessible_cmds
